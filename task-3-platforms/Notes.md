## HackerRank Experience
For the technical part of this module, I completed a coding challenge on HackerRank. This was a new experience for me because I had to write code that passed specific "hidden" test cases. It taught me that in professional programming, your code doesn't just need to "work"—it needs to be efficient and handle different types of inputs correctly.

## Google Forms for Data Collection
I created a 5-question Digital Literacy Awareness Quiz using Google Forms. My goal was to see how much my batchmates actually know about things like Two-Factor Authentication (2FA) and identifying phishing emails. It was interesting to see how a simple form can gather data from many people at once.

## Cloud Collaboration
One of the biggest takeaways from this task was seeing how Google Forms automatically syncs with Google Sheets. Being able to see responses update in real-time showed me how useful these cloud tools are for group projects where everyone needs to see the same data without sending files back and forth.

## Problem-Solving Skills
Doing the HackerRank challenge helped me practice the "logical thinking" side of B.Tech. It made me realize that digital literacy isn't just about using apps, but also about understanding the logic behind how software and data work.
